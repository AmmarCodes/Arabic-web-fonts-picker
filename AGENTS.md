# Agent Guidelines

## Tech Stack

- **Framework**: React + TypeScript (Vite)
- **Styling**: Tailwind CSS v4
- **UI Library**: coss.com/ui
- **Package Manager**: Bun

## Workflow

- **Commits**: Make small, frequent git commits for every step to ensure easy rollback.
- **Design**: Prioritize rich aesthetics, modern design, and responsiveness.
